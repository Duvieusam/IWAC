# iwac

> Command line tool to create Watson Assistant Workspaces

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i iwac --save-dev
```

## Usage

```js
var iwac = require('iwac');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
